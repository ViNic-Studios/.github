# Team 9 
## Outline
TaskForge is a cross-platform productivity app built for iOS and Android devices that eliminates procrastination by incorporating game elements to a task management system.

## Objectives (benefit to customer, key things to accomplish, criteria for success)
- Make accomplishing real world tasks engaging by incorporating game elements for task completion
- Provide a easy to use task manager application that fits in customers daily routines
- Users must be able to quickly create and manage tasks
- Must maintain a balance of productivity and gameplay so that game mechanics encourage productivity rather than distract users

## Expected/anticipated architecture
- Mobile app programmed using Flame add-on to Flutter for both Android and iOS
- Monolithic backend REST Api programmed in go
- Postgres DB for application entity storage
- Authentication will be handled by an undetermined third-party service

## Anticipated risks (engineering challenges, etc.)
